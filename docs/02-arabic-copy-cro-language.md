# 02 - Arabic Copy and CRO Language

## Copy Tone

Arabic tone should be:

- Saudi-friendly, not overly formal.
- Premium but warm.
- Persuasive without sounding fake.
- Emotion-led first, science-backed second.

Use Modern Standard Arabic mixed with light KSA phrasing. Avoid slang that feels cheap.
